แดชบอร์ดสรุปข้อมูลผู้ป่วย

อัปโหลดไฟล์เพื่อสร้างรายงานสรุป — ไม่มีการเก็บข้อมูลไว้ในระบบ ข้อมูลอยู่ในเบราว์เซอร์ชั่วคราวเท่านั้น

ไฟล์:

อัปโหลดไฟล์ข้อมูลผู้ป่วยเพื่อดูสรุป

รองรับไฟล์ .csv และ .xlsx — ลากไฟล์มาวาง หรือกดปุ่มด้านล่าง

มาจาก Google ชีท? เปิดชีท → File → Download → เลือก Microsoft Excel (.xlsx) หรือ Comma Separated Values (.csv) แล้วอัปโหลดไฟล์ที่ได้ที่นี่
แถวแรกของไฟล์ต้องเป็นหัวตาราง (ชื่อคอลัมน์) — ระบบจะอ่านคอลัมน์ตามที่มีในไฟล์จริง ไม่จำกัดโครงสร้างตายตัว

ตารางข้อมูลทั้งหมด
รายงานนี้สร้างจากไฟล์ที่อัปโหลดในเซสชันนี้เท่านั้น ไม่มีการบันทึกลงระบบหรือส่งข้อมูลออกไปที่ใด — ปิดหน้านี้หรือรีเฟรช ข้อมูลจะหายไปทั้งหมด หากต้องการดูอีกครั้ง อัปโหลดไฟล์ใหม่

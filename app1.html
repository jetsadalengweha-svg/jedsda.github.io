01

ติดตั้ง SMPS และตรวจแรงดัน

  • ต่อ L, N, GND ของ 220V AC เข้า Input SMPS
  • วัดด้วยมัลติมิเตอร์ที่ขา Output → ต้องได้ +5V และ +12V
  • ต่อ GND ของทั้งสองชุดให้ร่วมกัน (Common Ground)
```
02

ต่อ TCR T5000 IR Sensor

  • ขา VCC → +5V
  • ขา GND → GND
  • ขา OUT → Pin 3 (IN+) ของ LM393
  • ติด Sensor ใต้ก๊อก หันหน้าลงด้านล่างรับมือผู้ใช้
03

ต่อวงจร LM393P Comparator

  • Pin 8 (VCC) → +5V
  • Pin 4 (GND) → GND
  • Pin 3 (IN+) ← OUT ของ Sensor
  • Pin 2 (IN−) ← ขา Wiper ของ VR-10K
  • Pin 1 (OUT) → R-220Ω → R-1K → Base ของ 2N2222
  • VR-10K: ขาสองข้างต่อระหว่าง +5V ↔ GND
04

ต่อ Transistor 2N2222

  • Base (B) ← R-1K ← LM393 Pin1
  • Emitter (E) → GND
  • Collector (C) → ขา Coil− ของ Relay
  • ต่อ 1N4007 Diode คร่อม Relay Coil (ขั้วกลับ) ป้องกัน Back-EMF
05

ต่อ Relay 5V

  • Relay Coil+ → +5V
  • Relay Coil− → Collector ของ 2N2222
  • COM → +12V Rail
  • NO (Normally Open) → สายบวก Solenoid Valve
06

ต่อ Solenoid Valve 12V

  • ขา + ของ Solenoid ← สาย NO ของ Relay
  • ขา − ของ Solenoid → GND
  • ติดตั้ง Valve บนท่อน้ำก่อนเข้าก๊อก
  • ทดสอบ: เมื่อ Relay ทำงาน → COM-NO ต่อกัน → 12V ไหลผ่าน Valve → เปิดน้ำ
07

ทดสอบและปรับจูนระบบ

  • เปิดไฟ ตรวจแรงดัน SMPS ก่อนทุกครั้ง
  • ยื่นมือใต้ Sensor → LED Indicator ของ Relay ต้องติด → น้ำไหล
  • หมุน VR-10K เพื่อปรับระยะตรวจจับ (ใกล้/ไกล)
  • ตรวจรอยรั่วท่อน้ำทุกจุด
  • ใส่วงจรในกล่อง IP55 กันน้ำและความชื้น
```
⚠️ ข้อควรระวังด้านความปลอดภัย

• ห้ามสัมผัสสาย 220V AC ขณะวงจรเปิด  |  ใส่ฟิวส์ 1A ก่อน SMPS เสมอ
• วงจรต้องอยู่ในกล่องกันน้ำ IP55+ เนื่องจากอยู่ใกล้แหล่งน้ำ
